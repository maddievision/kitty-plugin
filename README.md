## Kitty Advance plugin

AU, CLAP, and VST3 plugin for [Kitty Advance](https://codeberg.org/roses/kitty), a GBA sound driver with samples + PSG support.

Runs an instance of kitty using [libmgba](https://github.com/mgba-emu/mgba)'s GBA core and forwards MIDI messages to it.
